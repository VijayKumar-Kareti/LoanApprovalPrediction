About Loan Approval Prediction System

This banking-inspired web application predicts loan approval outcomes using a trained machine learning pipeline. It is designed to look like a real underwriting portal, with transparent reasoning, responsive display behavior, and professional visual styling.

Key capabilities
  • Preprocesses user inputs and categorical banking data.
  • Compares multiple classifiers and selects the best-performing model.
  • Shows probability, confidence score, and explanation.
  • Works with common banking fields like credit history, property area and loan term.
Technology Stack

Flask | Bootstrap 5 | Python | Scikit-Learn | Pandas | Joblib
