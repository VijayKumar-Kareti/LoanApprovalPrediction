# Loan Approval Prediction System

## Project Description
The Loan Approval Prediction System is a web application designed to predict whether a loan application should be approved or rejected based on various realistic banking parameters. Utilizing machine learning algorithms, the application provides insights into the factors influencing loan approval decisions, making it a valuable tool for both applicants and financial institutions.

## Features
- User-friendly interface for loan application submission
- Predicts loan approval status (Approved/Rejected)
- Displays probability and confidence score of the prediction
- Provides reasons for the prediction based on input features
- Responsive design for optimal viewing on various devices
- Error handling for invalid inputs
- Animated cards and loading spinner for enhanced user experience
- Dark mode option (bonus feature)
- Prediction history and ability to download predictions as PDF (bonus feature)
- Charts dashboard for loan approval statistics (bonus feature)

## Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend:** Python, Flask
- **Machine Learning:** Pandas, NumPy, Scikit-Learn, Joblib
- **Visualization:** Matplotlib, Seaborn

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/LoanApprovalPrediction.git
   ```
2. Navigate to the project directory:
   ```
   cd LoanApprovalPrediction
   ```
3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```
4. Run the application:
   ```
   python app.py
   ```
5. Open your web browser and go to `http://127.0.0.1:5000/`.
